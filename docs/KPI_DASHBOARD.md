# KPI DASHBOARD

## Primary KPIs
*   **Total Processed Messages:** Sum of all `messages` collection.
*   **AI Conversion Rate:** % of leads moved to "Paid" by AI.
*   **Average Response Time:** Time between `client` message and `ai` message.
*   **MRR (Monthly Recurring Revenue):** Tracked via subscriptions.

## Real-time Monitoring
*   Server Uptime (Cloud Monitoring).
*   Firestore Read/Write Quotas.
*   Gemini API Token consumption.
